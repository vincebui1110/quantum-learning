In the last module a qubit's state was a static object - a vector \(|\psi\rangle = \alpha|0\rangle + \beta|1\rangle\). This module is about doing things to that state. Every operation you can perform on a single qubit (short of measuring it) is a gate, and every gate is a particular kind of matrix. This lesson pins down exactly which matrices are allowed and how they act.

A gate is a 2×2 matrix

A single qubit's state is a two-component column vector, so anything that transforms it into a new state is a \(2\times 2\) matrix \(U\). Applying the gate is nothing more exotic than a matrix times a vector:

\[ U|\psi\rangle = \begin{pmatrix} u_{00} & u_{01} \\ u_{10} & u_{11} \end{pmatrix} \begin{pmatrix} \alpha \\ \beta \end{pmatrix} = \begin{pmatrix} u_{00}\alpha + u_{01}\beta \\ u_{10}\alpha + u_{11}\beta \end{pmatrix}. \]

The output is a new pair of amplitudes - a new state of the same qubit. That is the whole mechanism. The interesting question is: which matrices are physically legal?

The rule: gates must be unitary

Not every matrix is allowed. A valid gate must be unitary, meaning it satisfies

\[ U^{\dagger} U = I, \]

where \(U^{\dagger}\) is the conjugate transpose (transpose the matrix, then take the complex conjugate of every entry) and \(I\) is the identity. Unitarity is the quantum analogue of "rigid motion": it is exactly the condition that keeps the state normalized.

Why unitary?

A legal state has total probability \(|\alpha|^2 + |\beta|^2 = 1\). A gate must send legal states to legal states, so it must preserve that length. The length-preserving linear maps are precisely the unitary ones - so "gate" and "unitary matrix" mean the same thing.

Unitarity buys you reversibility

Because \(U^{\dagger}U = I\), the matrix \(U^{\dagger}\) is a genuine inverse of \(U\). So if a gate \(U\) takes \(|\psi\rangle\) to \(U|\psi\rangle\), applying \(U^{\dagger}\) afterwards recovers the original exactly:

\[ U^{\dagger}\big(U|\psi\rangle\big) = (U^{\dagger}U)|\psi\rangle = I|\psi\rangle = |\psi\rangle. \]

Every quantum gate is therefore reversible - no information is ever lost during gate operations. This is a sharp contrast with measurement, which collapses a state irreversibly, and with classical gates like AND, which throw information away. It is also the reason quantum circuits are always built from invertible steps.

Worked example

Take the gate \(U = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\) (this is the X gate, coming next lesson). First, is it unitary? Its conjugate transpose is itself, and

\[ U^{\dagger}U = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} = I. \ \checkmark \]

Now apply it to \(|0\rangle = \begin{pmatrix}1\\0\end{pmatrix}\):

\[ U|0\rangle = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\begin{pmatrix} 1 \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ 1 \end{pmatrix} = |1\rangle. \]

The gate turned \(|0\rangle\) into \(|1\rangle\). Notice the output is still normalized - unitarity guaranteed that in advance.

Drawing it: the circuit wire

Quantum algorithms are drawn as circuits. A single qubit is a horizontal wire read left to right, standing for the flow of time. A gate is a box sitting on the wire:

|ψ⟩  ──────[ U ]──────  U|ψ⟩

The state enters on the left, the gate acts, and the transformed state leaves on the right. When several boxes sit on one wire, the qubit passes through them in order - the subject of the last lesson in this module. For now the key picture is simple: a wire is a qubit over time, and a box is a unitary matrix doing its single job.

Note

"Unitary" for complex matrices plays the role that "orthogonal" plays for real ones: both preserve length. The dagger \(\dagger\) (conjugate transpose) is the complex version of the plain transpose used for real rotation matrices.

Key takeaways

  • A single-qubit gate is a \(2\times 2\) matrix \(U\); applying it is the matrix-vector product \(U|\psi\rangle\).
  • Gates must be unitary: \(U^{\dagger}U = I\), where \(U^{\dagger}\) is the conjugate transpose.
  • Unitarity preserves normalization, so legal states map to legal states.
  • Unitarity also makes every gate reversible, with inverse \(U^{\dagger}\) - unlike measurement, gates never lose information.
  • In a circuit, a qubit is a wire read left to right and a gate is a box on that wire.
