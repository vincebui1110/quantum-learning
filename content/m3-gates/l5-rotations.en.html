In the qubit module you met the Bloch sphere: every single-qubit state is a point on the surface of a unit sphere, with \(|0\rangle\) at the north pole and \(|1\rangle\) at the south. This lesson delivers the payoff of that picture. Every single-qubit gate is simply a rotation of the Bloch sphere - and that one sentence organizes everything you have learned in this module.

Gates are rotations

Since a gate is unitary, it preserves the length of the state vector, so it cannot push a point off the sphere. A length-preserving motion of a sphere that keeps the centre fixed is exactly a rotation. So every single-qubit gate corresponds to rotating the whole sphere about some axis by some angle. The gates you already know fit right in:

  • \(X\), \(Y\), \(Z\): rotations by \(180°\) about the x-, y-, z-axes.
  • \(S\), \(T\): rotations about the z-axis by \(90°\) and \(45°\).
  • \(H\): a \(180°\) rotation about the diagonal axis midway between x and z.

The three rotation gates

We can rotate by an arbitrary angle \(\theta\) about each axis. These continuous families are the rotation gates \(R_x(\theta)\), \(R_y(\theta)\), \(R_z(\theta)\):

\[ R_x(\theta) = \begin{pmatrix} \cos\tfrac{\theta}{2} & -i\sin\tfrac{\theta}{2} \\[4pt] -i\sin\tfrac{\theta}{2} & \cos\tfrac{\theta}{2} \end{pmatrix}, \qquad R_y(\theta) = \begin{pmatrix} \cos\tfrac{\theta}{2} & -\sin\tfrac{\theta}{2} \\[4pt] \sin\tfrac{\theta}{2} & \cos\tfrac{\theta}{2} \end{pmatrix}, \] \[ R_z(\theta) = \begin{pmatrix} e^{-i\theta/2} & 0 \\[4pt] 0 & e^{i\theta/2} \end{pmatrix}. \]

Each rotates the Bloch sphere by angle \(\theta\) about the named axis. Setting \(\theta = 0\) gives the identity in every case (no rotation), as it should.

Why the half-angle?

Notice the \(\theta/2\), not \(\theta\), inside the matrices. The qubit's state vector lives in a 2-dimensional complex space that double-covers the Bloch sphere: rotating the physical sphere by a full \(360°\) only brings the state vector halfway around (it returns with a minus sign; a full \(720°\) is needed to truly return). So a physical rotation by \(\theta\) shows up as \(\theta/2\) in the state.

Worked example

Rotate \(|0\rangle\) by \(\theta = \pi\) about the x-axis. With \(\cos\tfrac{\pi}{2} = 0\) and \(\sin\tfrac{\pi}{2} = 1\):

\[ R_x(\pi)|0\rangle = \begin{pmatrix} 0 & -i \\ -i & 0 \end{pmatrix}\begin{pmatrix} 1 \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ -i \end{pmatrix} = -i\,|1\rangle. \]

Up to the overall phase \(-i\) (a global phase, physically invisible), this is \(|1\rangle\). A \(180°\) rotation about the x-axis carries the north pole \(|0\rangle\) to the south pole \(|1\rangle\) - exactly what \(X\) does, confirming \(R_x(\pi) = -iX\).

Every single-qubit gate is a rotation

More is true than "the gates we know are rotations." Any single-qubit unitary can be written as a rotation about some axis \(\hat{n}\) by some angle - and, equivalently, decomposed into a short sequence of rotations about fixed axes, for example

\[ U = e^{i\gamma}\,R_z(\alpha)\,R_y(\beta)\,R_z(\delta) \]

for suitable angles (the "Z-Y-Z decomposition"), with \(e^{i\gamma}\) an irrelevant global phase. This is why a small, fixed toolbox of gates is enough.

Universal single-qubit gate sets

A set of gates is universal for one qubit if you can approximate any single-qubit rotation as closely as you like by stringing its gates together. Rotations about two different axes already suffice - for instance \(R_z\) and \(R_x\). In the discrete, hardware-friendly world, \(\{H, T\}\) is universal: alternating Hadamards and \(T\) gates can approximate any rotation to arbitrary precision. That is the deep reason a real quantum computer needs only a handful of gate types to run any algorithm.

Key takeaways

  • Every single-qubit gate is a rotation of the Bloch sphere about some axis - unitarity keeps the state on the sphere.
  • \(R_x(\theta), R_y(\theta), R_z(\theta)\) rotate by \(\theta\) about the x-, y-, z-axes, built from \(\cos\tfrac{\theta}{2}\) and \(\sin\tfrac{\theta}{2}\).
  • The half-angle appears because the state space double-covers the Bloch sphere.
  • Any single-qubit unitary decomposes into a few fixed-axis rotations; small sets like \(\{H, T\}\) are universal.
